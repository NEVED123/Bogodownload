Why Bogodownload?

Bogodownload is a clever way to download any file you could imagine, without even using the internet!

How does it work?

Bogodownload will generate a random binary file anywhere between 1KB and 10MB, with a randomized file extension. Therefore, there is a small (but not zero!) chance that the file you download will be exactly the file you want. (Guys I swear its not a virus trust me bro)
